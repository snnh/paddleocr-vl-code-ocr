# 数据集说明

本目录用于放置数据构建和质检相关脚本。公开仓库不直接发布完整训练数据和评估集图片。

## 公开内容

| 文件 | 说明 |
| --- | --- |
| `audit_dataset_quality.py` | 检查图片/文本配对、空标注、重复样本和明显格式问题。 |

## 数据提交方式

- 训练数据：按比赛规则不要求公开，训练数据构建方法见 `docs/训练数据构建报告.md`。
- 评估集：按比赛要求单独提交下载链接，不直接放入 GitHub 仓库。
- 测试索引格式：OpenAI/PaddleOCR-VL 风格 JSONL，每条包含用户提示词、人工真值和图片路径。

## 标注原则

- 只转写图片中可见文字。
- 保留代码符号、缩进、换行、路径、大小写和阅读顺序。
- 不补全不可见内容。
- 不加入解释、总结或 Markdown 包装。
